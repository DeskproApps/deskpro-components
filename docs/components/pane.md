Pane
====
Renders a generic page section which may be displayed or hidden.

```jsx
<div>
    <Pane>
        Pane one!
    </Pane>
    <Pane hidden={true}>
        Pane two!
    </Pane>
</div>
```

### Props

**hidden={bool}**  
Set to true to hide the pane.


### CSS
Adds `dp-pane` to the root element.
