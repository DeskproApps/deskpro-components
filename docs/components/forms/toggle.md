Forms/Textarea
==============
Renders a form toggle.

```jsx
<Toggle checked={true} />
```

### Props

**value={string}}**  
The toggle value.

**name={string}**  
The name of the toggle.

**checked={boolean}**  
The status of the toggle.

**onChange={function}**  
Triggered when the toggle value changes. Receives the value and name of the toggle as arguments.
